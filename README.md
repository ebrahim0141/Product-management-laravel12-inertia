.................................................Project setup procedure.................................

1. git clone
3. npm install
4. composer install
5. copy .env.example .env
6. create mysql database
7. php artisan key:generate
8. php artisan migrate
9. php artisan db:seed
10. npm run dev
11. php artisan serve
12. visit: http://localhost:8000/

.................................................Project setup procedure.................................

1. Unzip file
2. Copy into htdocs
3. copy .env.example .env
5. Create mysql database
6. php artisan key:generate
7. php artisan migrate
8. php artisan db:seed
9. npm run dev
10. php artisan serve
11. visit: http://localhost:8000/
